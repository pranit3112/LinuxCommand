Linux Commands By Pranit Shirole

Beginner Commands

ls

Lists directory contents.

Example: ls -l lists files in the current directory in long format, showing details.

cd

Changes the current directory.

Example: cd /path/to/directory

pwd

Prints the current working directory.

Example: pwd

cp

Copies files or directories.

Example: cp source_file destination_file

mv

Moves or renames files or directories.

Example: mv old_name new_name

rm

Removes files or directories.

Example: rm filename
Example: rm -r directory_name (Recursively removes a directory and its contents)

mkdir

Creates a directory.

Example: mkdir directory_name

rmdir

Removes an empty directory.

Example: rmdir directory_name

touch

Creates an empty file or updates the access and modification times of a file.

Example: touch filename

echo

Displays a line of text.

Example: echo "Hello, World!"

cat

Concatenates and displays file content.

Example: cat filename

more

Displays file content one page at a time.

Example: more filename

less

Similar to more, but allows backward movement in the file as well.

Example: less filename

head

Outputs the first part of files.

Example: head -n 10 filename (Displays the first 10 lines)

tail

Outputs the last part of files.

Example: tail -n 10 filename (Displays the last 10 lines)

man

Displays the manual page for a command.

Example: man ls

who

Shows who is logged on.

Example: who

Intermediate Commands

ps

Displays information about active processes.

Example: ps aux shows a detailed list of all running processes.

grep

Searches for patterns within text.

Example: grep "search_term" filename searches for "search_term" in the specified file.

find

Searches for files and directories within a directory hierarchy.

Example: find /path -name "filename" searches for files named "filename" under "/path".

du

Estimates file and directory space usage.

Example: du -sh /path displays the total disk usage of the directory "/path" in human-readable format.

df

Reports file system disk space usage.

Example: df -h shows disk space usage in human-readable format.

kill

Terminates a process by PID.

Example: kill 1234
Example: kill -9 1234 (Forcefully terminates the process)

top

Displays tasks and system resource usage in real-time.

Example: top

chmod

Changes file permissions.

Example: chmod 755 filename (Sets read, write, and execute permissions for the owner, and read and execute permissions for group and others)
Example: chmod u+x filename (Adds execute permission for the owner)

chown

Changes file ownership.

Example: chown user:group filename

alias

Creates a shortcut for a command.

Example: alias ll='ls -la'

unalias

Removes an alias.

Example: unalias ll

history

Shows the command history.

Example: history

clear

Clears the terminal screen.

Example: clear

env

Displays the environment variables.

Example: env

export

Sets environment variables.

Example: export VARIABLE_NAME=value

unset

Unsets environment variables.

Example: unset VARIABLE_NAME

source

Executes commands from a file in the current shell.

Example: source filename

which

Shows the full path of a command.

Example: which ls

uname

Displays system information.

Example: uname -a

uptime

Shows how long the system has been running.

Example: uptime

ping

Sends ICMP ECHO_REQUEST packets to network hosts.

Example: ping hostname

Advanced Commands

netstat

Displays network connections, routing tables, interface statistics, masquerade connections, and multicast memberships.

Example: netstat -an | grep ESTABLISHED lists all established connections.

ifconfig

Displays or configures a network interface.

Example: ifconfig

iwconfig

Displays or configures wireless network interfaces.

Example: iwconfig

mount

Mounts a file system.

Example: mount /dev/sdX /mnt

umount

Unmounts a file system.

Example: umount /mnt

fdisk

Manipulates disk partition tables.

Example: fdisk -l

cronjob

Schedules commands to run at specific times.

Example: crontab -e edits the current user's cron jobs.

tar

Archives multiple files into a single file, often used with compression.

Example: tar -czvf archive.tar.gz file1 file2 creates a compressed archive "archive.tar.gz" containing "file1" and "file2".

gzip

Compresses files.

Example: gzip filename

gunzip

Decompresses files.

Example: gunzip filename.gz

zip

Compresses files into a ZIP archive.

Example: zip archive.zip file1 file2 creates a ZIP file named "archive.zip" containing "file1" and "file2".

unzip

Extracts files from a ZIP archive.

Example: unzip archive.zip

scp

Securely copies files between hosts.

Example: scp file user@remote_host:/path/to/destination

rsync

Efficiently transfers and synchronizes files between locations.

Example: rsync -avz source/ destination/

iptables

Configures the Linux kernel firewall.

Example: iptables -L lists all rules in the default filter table.

systemctl

Controls the systemd system and service manager.

Example: systemctl start|stop|restart|status service_name

journalctl

Views logs collected by the systemd journal.

Example: journalctl -u service_name

strace

Traces system calls and signals.

Example: strace -p 1234 (Traces the process with PID 1234)

lsof

Lists open files.

Example: lsof | grep filename

nc (netcat)

Reads and writes data across network connections.

Example: nc -l -p 12345 (Listens for incoming connections on port 12345)

awk

Pattern scanning and processing language.

Example: awk '{print $1}' filename prints the first column of each line in "filename".

sed

Stream editor for filtering and transforming text.

Example: sed 's/old/new/g' filename replaces all occurrences of "old" with "new" in "filename".

vim

Advanced text editor.

Example: vim filename

ssh

Opens a secure shell session to a remote host.

Example: ssh user@remote_host

telnet

Used for bidirectional interactive text-oriented communication over a network (often replaced by more secure methods like SSH).

Example: telnet hostname port connects to a specified host and port.

dd

Converts and copies files.

Example: dd if=/dev/sdX of=/path/to/image.img

wget

Downloads files from the web.

Example: wget http://example.com/file.zip

curl

Transfers data from or to a server using various protocols.

Example: curl http://example.com

nano / vim / emacs

Text editors.

Example: nano filename
Example: vim filename
Example: emacs filename

cd Command Usage

Basic Usage

Change to the specified directory:

Example: cd /home/user/Documents

Home Directory

Change to the home directory:

Example: cd
Another way to change to the home directory: cd ~

Parent Directory

Change to the parent directory:

Example: cd ..

Root Directory

Change to the root directory:

Example: cd /

Previous Directory

Change to the previous directory:

Example: cd -
Usage: cd /home/user/Documents
cd /var/log
cd - (This will take you back to /home/user/Documents)

Relative Path

Change to a directory relative to the current directory:

Example: cd ./Downloads
Example: cd ./Pictures

Combining with Other Commands

Change to the specified directory and execute a command:

Example: cd /var/log && ls
Example: cd /tmp && touch example.txt

Environment Variables

Change to the home directory using the HOME environment variable:

Example: cd $HOME
Example: cd $HOME

Special Characters

Use the physical directory structure without following symbolic links:

Example: cd -P /path/to/symlink
Example: cd -P /var/www/html

Using Wildcards

Change to directories using wildcard characters:

Example: cd /var/l*
Example: cd /usr/l*

netstat Command Usage

Display Network Connections

Displays network connections, routing tables, interface statistics, masquerade connections, and multicast memberships.

Example: netstat -an lists all network connections and listening ports.

Filter Established Connections

Lists all established connections.

Example: netstat -an | grep ESTABLISHED

Display Routing Table

Shows the routing table for the system.

Example: netstat -r

chmod Command Usage

Change File Permissions

Sets permissions for a file or directory.

Example: chmod 755 filename (Sets read, write, and execute permissions for the owner, and read and execute permissions for group and others)

Add Execute Permission

Adds execute permission for the owner of the file.

Example: chmod u+x filename

Remove Write Permission

Removes write permission for the group.

Example: chmod g-w filename

Set Permissions Recursively

Applies permissions recursively to directories and their contents.

Example: chmod -R 755 directory_name
