Linux Commands BY PRANIT SHIROLE

Beginner Commands

ls

Lists directory contents.

Example: ls -l lists files in the current directory in long format, showing details.

cd

Changes the current directory.

Example: cd /path/to/directory

pwd

Prints the current working directory.

Example: pwd

cp

Copies files or directories.

Example: cp source_file destination_file

mv

Moves or renames files or directories.

Example: mv old_name new_name

rm

Removes files or directories.

Example: rm filename
Example: rm -r directory_name (Recursively removes a directory and its contents)

mkdir

Creates a directory.

Example: mkdir directory_name

rmdir

Removes an empty directory.

Example: rmdir directory_name

touch

Creates an empty file or updates the access and modification times of a file.

Example: touch filename

echo

Displays a line of text.

Example: echo "Hello, World!"

cat

Concatenates and displays file content.

Example: cat filename

more

Displays file content one page at a time.

Example: more filename

less

Similar to more, but allows backward movement in the file as well.

Example: less filename

head

Outputs the first part of files.

Example: head -n 10 filename (Displays the first 10 lines)

tail

Outputs the last part of files.

Example: tail -n 10 filename (Displays the last 10 lines)

man

Displays the manual page for a command.

Example: man ls
